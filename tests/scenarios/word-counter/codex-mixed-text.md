# Word Counter - Codex - Mixed Text

## Scenario

Use the `word-counter` skill in `Codex` to count a short Chinese-English mixed sentence.

## Input

`统计这段中英混排文本的字数：你好 OpenAI 2026`

## Expected Behavior

- The skill is selected successfully.
- The output is concise.
- The result includes a total count.
- The output follows Chinese language conventions.

## Acceptance Criteria

- The returned count is stable and deterministic.
- The output format matches the skill contract.
- No extra unsupported statistics are shown unless requested.
