# Android Interview - Claude Code - MVP Session

## Scenario

Run the Android interview MVP session in a candidate self-practice flow with immersive English and TTS enabled.

## Expected Behavior

- The session runner reads JD, resume, answer fixtures, and question bank files.
- The runner writes local session artifacts.
- The session includes `session.json`, `score.json`, `transcript.md`, and `report.html`.
- The session covers multi-question technical rounds and includes `round3`, with generated fallback questions when the bank does not fully cover a round.
- When `edge-tts` is installed, TTS audio files are generated under `tts/`.
